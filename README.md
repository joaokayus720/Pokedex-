# Pokédex 

Pokédex web com busca, favoritos e detalhes de Pokémon usando a PokeAPI.

## Funcionalidades

- Listagem paginada (12 por página)
- Botões anterior/próxima
- Busca por nome com sugestões automáticas
- Clique no card: exibe imagem, nome, ID, tipos, altura e peso
- Favoritar Pokémon (salva no localStorage)
- Botão para ver apenas favoritos
- Mensagens de carregando, erro e nenhum resultado
- Layout responsivo

## Como usar

1. Baixe os arquivos (`index.html`, `style.css`, `script.js`)
2. Abra o `index.html` no navegador

## Tecnologias

- HTML, CSS, JavaScript (puro)
- Bootstrap 4 (modal + grid)
- PokeAPI

## Personalização

- Altere `limit` no `script.js` para mudar quantos Pokémon aparecem por página.
- Modifique as cores no `style.css`.
